1. **Si sos nuevo con estas funcionalidades:**

2. **Si necesitás resolver algo rápido:**
   - Consultá la Guía Rápida para acciones específicas
   - Buscá en Ejemplos Prácticos situaciones similares a la tuya

3. **Si querés entender el funcionamiento interno:**
   - Revisá los Detalles Técnicos para comprender la implementación
